## What I'm Exploring

- Building tiny tools that remove repeated friction from daily development.
- Agentic engineering workflows: where humans should decide, where agents should run, and how to make handoffs visible.
- Product ideas that are useful before they are polished.
- Education-oriented software that helps people understand systems instead of just consuming outputs.

## Philosophy

Ship small, learn fast, and keep the tool close to the actual problem.
